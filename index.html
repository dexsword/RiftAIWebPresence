Meta Snapshot
See the top legends, most-played cards, and archetype breakdown for Set 2 (current) or Set 3 (preview). Filtered to legal decks only, sourced from tournament players.

Deck Search & Archetypes
Search by legend, card name, or archetype. Get shell breakdowns (e.g. Draven Kai'Sa vs Draven Rhasa), anchor percentages, quality scores, and legal example decklists.

Card Synergy Engine
Find what cards combo best with any card in the game. Powered by vector similarity on card text embeddings + real co-occurrence data from thousands of decklists.
Live

Deck Builder
Generate a statistically optimized 40-card deck for any legend. Weighted by gold/silver tier tournament data. Falls back to synergy engine when data is thin.

🔧 How it was built

RiftAI is a solo project built from scratch — Node.js, Discord.js, PostgreSQL with pgvector, SQLite, and OpenAI APIs. The harvester pulls decklists from Piltover Archive every 6 hours, classifies them by format and quality tier, and feeds a vector-similarity synergy engine backed by card embeddings.

📍 Where it's played

I play at Vegas Singles Trading Cards in Las Vegas, NV. RiftAI was originally built to help my local community understand the meta better — and grew from there into something more comprehensive.

🗺️ What's next

Active development. Next priorities: web dashboard, counter builder, Set 3 card support when Unleashed launches, and a premium tier for deeper analysis tools. The dataset grows every 6 hours.

Privacy & Terms

RiftAI collects minimal data and is not affiliated with Riot Games or the official Riftbound product.

Last updated: April 2, 2026

RiftAI ("the Bot", "we", "us") is a Discord bot built and operated by Dexsword as an independent community project. This Privacy Policy explains what data we collect, how we use it, and your rights.

Data We Collect

When you use RiftAI in a Discord server, we may collect:

  • Discord User ID — to track usage limits and tier access
  • Discord Server (Guild) ID — to configure server-level settings
  • Command inputs — solely to process your request and return a response
  • Usage counts — to enforce free tier daily limits

We do not collect your Discord username, email address, message history, or any data outside of direct bot interactions.

Data We Do Not Collect

RiftAI does not collect, store, or process: real names, email addresses, payment information, location data, browsing history, or any data from channels the bot is not actively responding in.

How We Use Your Data

Data is used exclusively to operate the bot — enforce rate limits, return personalized responses, and improve features. We do not sell, share, or monetize your data with any third parties.

Third-Party Services

RiftAI uses the OpenAI API to generate card explanations and quality scores. Command inputs related to those features are processed by OpenAI subject to their privacy policy. Deck data is sourced from Piltover Archive (piltoverarchive.com), a public community resource.

Data Retention

Usage counters reset daily. No persistent user profiles are maintained beyond what is necessary to operate tier access. You may request deletion of your data at any time by contacting us via Discord.
